# 创建文章指南

## 方式一：使用脚本（推荐）

```bash
./new-post.sh
```

**特点：**
- 交互式输入，中文提示
- 支持选择 Post（首页显示）或 Programming（知识库）
- 自动填充日期、标签、分类

**示例：**
```
选择内容区域:
  1) Post        - 博客文章 (显示在首页)
  2) Programming - 技术文档 (独立知识库)
请选择 (1/2, 默认: 1): 1

文章标题: 我的投资心得
分类: Life/投资
文件名: investment-2026
标签: 投资,理财
分类标签: 生活,财务
是否为草稿? (y/n): y
```

---

## 方式二：Hugo 命令

```bash
# 使用默认模板
hugo new content/Post/Life/my-article.md

# 使用 programming 模板（适合技术文档）
hugo new --kind programming content/Programming/Git/git-rebase.md
```

**特点：**
- Hugo 官方标准方式
- 基于 archetype 模板自动生成
- 适合快速创建

---

## 方式三：手动创建

直接创建 `.md` 文件，手动填写 frontmatter：

```markdown
---
title: 文章标题
date: 2026-02-12
draft: true
tags: []
categories: []
---

正文内容...
```

---

## 内容区域说明

### Post 目录
- **显示位置**：博客首页、归档页
- **URL 格式**：`/p/:slug/`
- **用途**：日常博客、生活记录、动态更新

### Programming 目录
- **显示位置**：不在首页（独立访问）
- **URL 格式**：`/programming/:slug/`
- **用途**：技术文档、知识库、教程

---

## Archetype 模板

可在 `archetypes/` 目录自定义模板：

- `default.md` - 默认模板
- `programming.md` - 技术文档模板（已配置）

模板变量：
- `{{ .Name }}` - 文件名
- `{{ .Date }}` - 创建日期
- `{{ replace .Name "-" " " | title }}` - 格式化标题

---

## 常用命令

```bash
# 预览（包含草稿）
hugo server -D

# 构建
hugo

# 查看版本
hugo version
```
