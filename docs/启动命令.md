# Hugo 启动命令

## 开发服务器

```bash
# 基础启动（不含草稿，模拟正式环境）
hugo server

# 基础启动（含草稿）
hugo server -D

# 禁用缓存启动（SCSS 样式改了不生效时用这个）
hugo server -D --ignoreCache

# 禁用缓存 + 禁用浏览器 HTTP 缓存（样式/JS 改了浏览器还是用旧版时加上）
hugo server -D --ignoreCache --noHTTPCache

# 指定端口
hugo server -D --port 1314
```

> 开发服务器默认地址：http://localhost:1313/blog-hugo-stack/

---

## 构建静态文件

```bash
# 正常构建（输出到 public/ 目录）
hugo

# 构建包含草稿的内容
hugo --buildDrafts

# 构建并忽略缓存（遇到样式没更新时用）
hugo --buildDrafts --ignoreCache
```

---

## 缓存相关

| 参数 | 作用 |
|---|---|
| `--ignoreCache` | 忽略 Hugo 的 SCSS/资源编译缓存 |
| `--noHTTPCache` | 禁止浏览器缓存静态资源 |
| `--gc` | 构建后清理无用的缓存文件 |

**手动清除缓存：**
```bash
rm -rf resources/
```

## 其他常用命令

```bash
# 查看 Hugo 环境信息
hugo env

# 查看版本
hugo version

# 新建文章（使用模板）
hugo new content/Post/<分类>/<文章名>.md
```
