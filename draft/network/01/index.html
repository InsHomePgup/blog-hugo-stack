      network Lan :Local Area Network Wan : Wide Area Network

      简单理解，wan用于外网，lan用于内网

      什么是无线AP？ 无线AP，无线接入点，用于无线网络的交换机。

      路由模式，光猫进行光电转换+拨号上网的功能。 桥接模式，光猫只进行光电转换，拨号上网的功能交给主路由进行。

      AC，是主控制器（Access Controller）的缩写，可用来管理、控制各AP，让各个AP接入互联网。

      AP，是接入点（Access Point）的缩写，路由器、AP面板等等都可以是一个AP。
