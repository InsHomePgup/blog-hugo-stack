##### 修改远程仓库的名称
```shell

git remote rename [oldName] [newName]

git remote rename origin github
```

##### 查看远程仓库的信息

``` shell

git remote show origin

```

##### 修改远程仓库的地址

```
git config remote.origin.url
```
